# Nexus AI Roblox Studio Plugin

Connect Roblox Studio directly to the [Nexus AI Platform](https://nexus-ai.com).

## Features
- ⚡ AI Script Generation from Studio
- 📦 Asset Import from Nexus Marketplace  
- 🔄 Code Sync between platform and Studio
- 📚 Snippet Library

## Installation

1. Download `NexusAI.lua` from this repo
2. Open Roblox Studio
3. Go to **Plugins → Manage Plugins → Install from File**
4. Select the downloaded `NexusAI.lua` file
5. Click the **Nexus AI** button in the toolbar
6. Enter your API Key from the Nexus AI Platform (Roblox AI → Studio Integration)

## Get Your API Key
Go to **Nexus AI Platform → Roblox AI → Studio Integration** and generate your key.
